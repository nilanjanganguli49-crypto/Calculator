# Project Overview

This project is a digital calculator application built with Flutter. It provides a user-friendly interface for performing basic and advanced arithmetic calculations.

# Features

*   **Basic Arithmetic Operations:** The calculator supports addition, subtraction, multiplication, and division.
*   **Square Root:** A "âˆš" button to calculate the square root of the current number.
*   **Delete Function:** A "DEL" button to delete the last entered digit.
*   **Clear Function:** A "C" button is available to clear the current input and reset the calculator.
*   **Responsive UI:** The layout is designed to adapt to different screen sizes.

# Design

*   **Color Scheme:** The calculator uses a simple and clean color scheme with a white background and black text. Operator buttons are highlighted with an orange color, the clear button with red, the delete and square root buttons with orange, and the equals button with green.
*   **Layout:** The calculator has a traditional layout with the display at the top and the buttons arranged in a grid below.
*   **Typography:** The app uses the default font provided by Flutter.
